Swipp at a glance

Swipp is a modern cryptocurrency based on a secure hashing algorithm - offering blazingly fast anonymous transactions. The code base is actively developed by a skilled software engineering team with decades of experience.

The currency is a long-term project with a long-term development plan. The currency features sane returns and specifications intended to create a healthy cryptocurrency with longevity.

X11

Swipp uses blockchain technology and is based on the X11 hashing algorithm in order to secure transactions and mining of the currency.

Masternodes

Swipp offers masternodes, securing and improving the speed of the network. With a collateral of 30 000 Swipp, anybody can run a masternode and receive a part of the Swipp generated on the network.

Roadmap

We have a preliminary roadmap subject to change as we develop Swipp and it's infrastructure.

Q4 2017

  • Takeover announcement.
  • Initial financing.
  • Publish website.
  • Publish block explorer.
  • Update wallet and daemon source code.
  • Post announcement on bitcointalk.
  • Rehaul some of the source code that seems to be in pretty rough shape.
  • Add support for additional exchange.

Q1 2018

  • Publish web wallet (with mobile support).
  • Add donation-driven bounty system to the web site.
  • Slim down and improve the desktop wallet, making it more user-friendly.
  • Implement forum on the website.

Q2 2018

  • Agreement with a Swedish business partner, offering co-location, storage, server leasing and VPS rentouts. They will accept Swipp as a payment option.
  • Implement financial data into the wallet (graphs showing price development and other good stuff)

Q3 2018

  • Submit Swipp to Bittrex and/or C-CEX.
  • Partnership to provide a Swipp VISA debit card. This service can provide both virtual and plastic VISA debit cards. Deposit Swipp and pay for services with it - anywhere in the world.

Q4 2018

  • Submit Swipp as a base currency to an exchange. If this fails, implement something similar to the originally planned Swipp exchange plattform.
